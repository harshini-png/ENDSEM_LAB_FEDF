# Student Dashboard Login & Registration System

## Introduction
The Student Dashboard Login & Registration System is a web-based application developed using HTML, CSS, and JavaScript. The system allows students to register, log in, and access their personal dashboard through a simple and user-friendly interface.

## Objectives
- To provide a secure student registration system.
- To enable students to log in using their credentials.
- To display a personalized dashboard after successful login.
- To demonstrate front-end web development concepts.

## Features
- Student Registration
- Student Login
- Dashboard Interface
- Form Validation
- Responsive Design
- Logout Functionality

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Project Modules

### 1. Registration Module
Allows new students to create an account by entering:
- First Name
- Last Name
- Email
- Password
- Date of Birth
- Phone Number
- Region

### 2. Login Module
Allows registered students to log in using their email and password.

### 3. Dashboard Module
Displays:
- Welcome Message
- Student Information
- Navigation Options
- Logout Button

## How to Run

1. Download or clone the project.
2. Open the project folder.
3. Open `index.html` in a web browser.
4. Register a new account.
5. Log in using the registered credentials.
6. Access the student dashboard.
